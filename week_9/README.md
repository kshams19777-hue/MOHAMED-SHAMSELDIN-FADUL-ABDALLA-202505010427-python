# Ticket Registration System - Week 9 Tutorial

## Purpose of the Application
The IT Helpdesk Ticket Registration System is a modular Python application designed for the IT Support team at City University Malaysia. It allows students to log technical issues (e.g., LMS login failure, WiFi issues) and automatically assigns a technician based on the issue's priority level.

## Tech Stack
* **Language:** Python 3
* **Concepts Used:** Modular Programming, Functions, Conditional Logic, Dictionaries
